Always try to understand and learn from other peoples code you are using.

  • Think
  • Discuss
  • Contribute
- You are saving time by using someone elses code, you have a few minutes to invest in some learning
- Look under the hood, beyond what docs and examples show (not all at once), IDE ctrl-click
- Try to think of ways to improve it (even if you don't actually modify it)
- Try to identify any design patterns used
- If you run into any issues, contribute
